# Integração RD Station

Este projeto foi construído para ser utilizado no sistema de CRM que trabalho atualmente, pela empresa Benner: https://www.benner.com.br 

Ele possui um modelo de envio de contatos criado conforme a necessidade da empresa.

Atualmente utiliza o HttpClient para executar os métodos de GET para consultar determinado o token de acesso conforme os dados do cliente e um método POST para enviar os contatos para a plataforma da RD Station.
